Holusion content manager est un service qui vous permet de créer et modifier en direct le contenu de votre application Ipad Holusion Companion.

Ajouter de nouvelles pages, créer des liens entre elles, les trier par catégories, suivez notre guide pour maitriser en profondeur notre interface.

Sommaire

  • Bien démarrer
  • Créer et modifier votre page
  • Créer des liens entre les pages

Bien démarrer

Tout d'abords, assurez vous que l'intreface soit dans la bonne langue.

Pour celà, cliquez sur la flèche à côté de votre nom et selectionnez "FR".

{% picture grid "img/documentation/companion/app_navbar.png" class="img-fluid" --alt capture d'écran de la barre de navigation de holusion content manager %}

L'interface se compose de 4 parties.

Pages

Ajoutez et éditez les pages de votre application

Paramètres

Retrouvez les informations générales de l'application

Médias

Ajoutez des images et vidéos à votre projet

Thème

Customisez l'apparence globale de votre application

Premiers pas

{% picture grid "img/documentation/companion/medias.jpg" class="mb-4" --alt capture d'écran de l'onglet medias de l'application holusion content manager %}

Ajouter des médias

Avant de créer vos pages, commencez par aller dans l'onglet "médias" pour ajouter les vidéos projetées sur votre borne holographique, ainsi que les images qui seront présentes sur les pages de votre application.

{% picture grid "img/documentation/companion/new_page.jpg" --alt capture d'écran de la création d'une nouvelle page dans holusion content manager %}

Créer votre première page

Chaque page correspond à une vidéo affichée sur la borne holographique. Il faudra donc autant de page que de vidéo à projeter

Dans l'onglet "pages", entrez le nom de votre nouvelle page puis cliquez sur le bouton "créer".

Éditer votre page

En cliquant sur la carte correspondante à votre page, vous avez accès à différents onglets : apperçu, édition et liens.

{% picture grid "img/documentation/companion/preview.jpg" class="mb-4" --alt capture d'écran de l'onglet prévisualisation de page de l'application holusion content manager %}

Apperçu

La section apperçu nous permet d'avoir une vision globale de la page, avec à gauche ses informations ainsi que la vidéo qui sera projetée sur la borne holographique, et à droite son apperçu sur l'application Holusion Companion.

{% picture grid "img/documentation/companion/edit_left.jpg" class="mb-4" --alt capture d'écran de l'onglet prévisualisation de page de l'application holusion content manager %}

Édition

L'inglet "édition" nous permet de modifier ces informations.

On retrouve à gauche les informations de l'apperçu que l'on peut alors modifier.

  • La miniature qui sera affichée en prévisualisation de la page
  • La vidéo projetée sur la borne holographique préalablement ajoutée dans les médias de l'application
  • La catégorie dans laquelle est rangée la page. (voir Gestion des catégories)

{% picture grid "img/documentation/companion/edit_right.jpg" class="mb-4" --alt capture d'écran de l'onglet prévisualisation de page de l'application holusion content manager %}

A droite, on peut modifier directement le contenu de la page en cliquant sur l'image pour la modifier et grâce à un éditeur de texte au format markdown.

Pour une application plus personnelle, vous pouvez intégrer du contenu dans une image en 2160x1520px qui prendra la totalité de l'écran, et faire des liens personnalisés avec notre éditeur de lien présenté plus bas.

{% picture grid "img/documentation/companion/links.jpg" class="mb-4" --alt capture d'écran de l'onglet liens de l'application holusion content manager %}

Liens

Navigez entre vos pages grâce à l'outil de création de liens.

Choisissez entre bouton, image et forme personnalisée puis éditer vos liens afin de personnaliser entièrement votre application.
